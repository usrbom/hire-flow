# Build AI Product Sense

This repository is a personal career-operations workspace centered on internship and early-career applications in product, product marketing, TPM, program management, and adjacent roles.

## What this repo is for

- managing application workflows in Markdown
- storing agent instructions and resume-tailoring rules
- organizing tasks and role-specific analysis
- generating tailored application folders under `tailored/`

## Public vs private content

This GitHub version is intentionally sanitized for collaboration.

Included:
- workflow and process documentation
- folder structure
- placeholder files that preserve expected paths
- local document-generation scripts

Excluded:
- personal base resumes
- archived resume variants
- tailored application content
- private reference resume PDFs

## Canonical structure

- `AGENTS.md`: workspace-level operating rules
- `Knowledge/my-resumes/AGENTS.md`: resume-tailoring workflow
- `Knowledge/my-resumes/base/`: local-only base resumes and index
- `tailored/<Company>/<Role>/`: local-only tailored application outputs
- `Tasks/`: general task tracking

`tailored/` at the repository root is the only canonical output location for tailored applications.

## Collaboration notes

If multiple people contribute to this repo:

- keep all workflow changes in Markdown
- preserve the existing folder structure
- do not commit personal resume files or generated application folders
- use placeholder `README.md` files to document any ignored directories
